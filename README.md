# payroll-processing-excel-project
Excel-based payroll system with payslip generator and dashboard
Payroll Processing & Analysis (Excel Project)
🔹 Overview

This project simulates a real-world payroll processing system using Microsoft Excel. It includes salary computation, statutory deductions, attendance-based adjustments, and automated payslip generation.

🔹 Features
Gross salary calculation (Basic + HRA + Allowance)
Attendance-based salary adjustment
PF calculation (12% of Basic Salary)
TDS calculation using IF conditions
Net salary computation
Automated payslip generator using VLOOKUP
Data validation and error handling
Interactive dashboard for payroll insights
🔹 Tools Used
Microsoft Excel
Functions: IF, VLOOKUP, Pivot Tables, Data Validation
🔹 Project Structure
Payroll Sheet – Salary calculations
Payslip Sheet – Dynamic payslip generator
Dashboard – Data visualization
🔹 Outcome

Developed a structured payroll system ensuring accuracy, automation, and real-world applicability.
